# namemash

> Generate a list of possible usernames from a person's first and last name.

---

## Usage

```bash
python3 namemash.py $PATH_TO_FILE_OF_FIRST_AND_LAST_NAMES
```

Example file of first and last names:

```txt
Steve Rogers
Tony Stark
Peter Parker
```
